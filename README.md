# 💻 Amazon Laptop Sales Dashboard

This project analyzes **Amazon laptop sales data** using **Power BI** to provide insights into laptop pricing, sales performance, ratings, and stock availability. The dashboard helps visualize key metrics and trends to understand product performance and market patterns.

## 📊 Project Overview

The dataset contains information about laptops listed on Amazon, including:

* Brand
* Model
* Screen Size
* RAM
* Hard Disk
* CPU
* Graphics
* Operating System
* Rating
* Price
* Sale Product Count
* Total Sales
* Available Stock

Using this data, an interactive **Power BI dashboard** was created to analyze laptop sales and product performance.

## 📈 Dashboard Features

The Power BI dashboard provides the following insights:

* 💰 **Total Sales Analysis**
* 🏷 **Laptop Price Distribution**
* ⭐ **Product Ratings Analysis**
* 🖥 **Brand-wise Laptop Comparison**
* 📦 **Available Stock Overview**
* 📊 **Sales Performance by Product**
